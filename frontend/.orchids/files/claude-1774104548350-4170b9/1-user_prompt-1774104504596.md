Headers: { "Content-Type": "application/json", "X-API-Key": API_KEY }
Body: { "question": string, "conversation_id": string | null }
Response: { success: true, data: { question, sql, rows, count, conversation_id } }

If API call succeeds: stream the response text word by word, render table if rows exist, show SQL from data.sql

"cgpa"/"section"/"performance" → section-wise comparison table
